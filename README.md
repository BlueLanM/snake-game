# 魔改贪吃蛇 🐍

魔改贪吃蛇游戏

## 🎮 操作指南

| 操作 | 按键 |
|------|------|
| 移动 | `↑↓←→` 或 `WASD` |
| 加速 | 按当前方向键 |
| 射击 | `J` 或 `X` |
| 重启 | `空格` |

## ✨ 游戏特性

- 🐍 **流畅的蛇移动** - 方向队列系统，支持快速连续转向
- ⚡ **加速机制** - 按当前方向键可加速移动
- 🔫 **射击系统** - 消耗身体长度发射子弹
- 👹 **敌人系统** - 随机生成的敌人，增加挑战性
- 🎯 **战斗奖励** - 击杀敌人获得+2身体长度和+5分数
- 🎨 **精美视觉效果** - 渐变色彩、粒子特效、动画效果
- 🏆 **最高分记录** - 使用 localStorage 永久保存最高分
- 🔵 **淡蓝色食物** - 吃掉增加弹药

## 🎯 游戏规则

### 基础规则
- 控制蛇吃食物（淡蓝色圆点）来增加长度和分数
- 每吃一个食物，蛇的速度会逐渐加快
- 蛇可以穿墙（从一边出去，另一边进来）
- 撞到自己的身体或敌人会导致游戏结束

### 弹药系统
- **弹药计算**：弹药数 = 蛇身长度 - 3
- **最低长度**：蛇长度为3时无法射击
- **消耗机制**：每次射击消耗1格身体长度

### 敌人系统
- 每3秒生成一个敌人（最多5个）
- 敌人会随机移动
- 触碰敌人 = 游戏结束
- 用子弹击杀敌人获得奖励
